Changelog

All notable changes to Flashy — the J2534-style Pass-Thru interface for the Adafruit Feather M4 CAN Express — are documented here.

Format follows Keep a Changelog. Tags: Added Changed Fixed Removed Infra

v1.4.0 — First Release 2026-04-22

Added First public release of Flashy — a clean-room, MIT-licensed J2534-style Pass-Thru tool. Built from public protocol documentation and reference manuals only. No commercial flash-tool source code, kernels, or proprietary IP referenced or included.

Hardware

  • Target: Adafruit Feather M4 CAN Express (ATSAME51J19A, 120 MHz Cortex-M4F, built-in CAN peripheral + transceiver)
  • Optional Adalogger FeatherWing (SD card + PCF8523 RTC) for on-device .bin storage and timestamped filenames
  • Annotated wiring photos and bench-pinout reference for GM modules under docs/

Firmware features

  • Auto-detect CAN at boot — sweeps 500k / 250k / 1M / 125k baud, no INIT required for normal operation
  • Interactive menu — six top-level options (Help / Connect / Diag / ECU / Hacker / RAW), per-ECU operation menus, capability-gated commands
  • UDS / ISO-TP / CAN driver stack — standard OBD-II services, multi-frame ISO-TP segmentation, interrupt-driven CAN RX with 4 KB ring buffer
  • OBD-II diagnostics — Read DTC ($03 with human-readable decode like P0300 / U0100), Clear DTC ($04), VIN read/write, OSID, ECU reset
  • SETCLOCK + RTC — PCF8523 wall clock, timestamped filenames (<OSID>-YYMMDD-HHMMSS.bin), session-counter fallback
  • SCAN — multi-module bus probe, returns module type + VIN where available
  • Bus Sniffer — live CAN frame display on serial, accept-all filter

Module support

  • E38 ECM (MPC561) — full read, cal/full write, kernel upload, VIN read/write, diag
  • E67 ECM (MPC565) — full read (verified, 21 KB/s), cal/full write, kernel upload, VIN read/write
  • E92 ECM (MPC5674F, e200z7 Book E) — clean-room kernel verified: full 4 MB read in 7 min 20 s at 9 KB/s, three consecutive checksum matches (0x233F8347)
  • T87 / T87A TCM (SPC564A80) — HS-CAN read/write, BAM-mode read/write (4 MB in 450 s, verified against JTAG readback)
  • T93 TCM (SPC564A80) — HS-CAN diagnostics, BAM read in progress
  • T42 TCM, E40 ECM — diagnostic-only at present

Clean-room kernels (Cernels)

  • Cernels/e92_read/ — 1816-byte Book E read kernel for MPC5674F. Handles $01 PING, $1A readDataByIdentifier, $23 ReadMemoryByAddress (multi-frame ISO-TP TX with full FC/BS/STmin support), $11 01 ECUReset (SIU SRCR). Inherits bootloader stack, silently discards unrecognized frames. Built with the free NXP powerpc-eabivle-gcc toolchain.
  • Cernels/t87a_read/ — VLE/Book E read kernel for SPC564A80, work-in-progress.
  • MIT licensed. Written from Freescale / ST reference manuals and FlexCAN register definitions only. Embedded copyright signature in each binary.

Host tools (Python & .exe)

  • capture_bus.py — raw CAN bus capture to SavvyCAN-format CSV
  • vin_scan.py / vin_update.py — multi-module bus scan + VIN updater
  • detect_port.py — Feather COM port auto-detect (USB VID match)
  • seed_key_algo.py — GM 2-byte seed/key (1280-algo lookup)
  • gm5byte/ — GM 5-byte seed/key (AES-128 + SHA-256)
  • bin2header.py — .bin → C header for embedding kernels in firmware
  • build_exe.py — PyInstaller build pipeline (portable dist/Flashy-Tool/ bundle, no Python required on target machine)

Documentation

  • docs/Read-Write-Instructions.html — step-by-step usage guide

Distribution
